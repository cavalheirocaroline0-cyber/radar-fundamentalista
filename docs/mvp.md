# MVP — Radar Fundamentalista

## O que é o MVP?

O MVP do Radar Fundamentalista será a primeira versão funcional do produto, com foco em resolver o problema principal: facilitar a comparação entre empresas brasileiras usando indicadores fundamentalistas.

Nesta primeira versão, o objetivo não é criar uma plataforma completa, mas sim validar se uma visualização simples, organizada e comparável já gera valor para o usuário.

---

## Problema que o MVP quer resolver

Investidores iniciantes e analistas em formação têm dificuldade para interpretar indicadores financeiros de forma prática.

Mesmo quando os dados estão disponíveis, eles costumam aparecer de maneira técnica, espalhada e pouco visual, dificultando a comparação entre empresas.

---

## Objetivo do MVP

Criar uma base inicial de empresas e um dashboard simples que permita:

* Visualizar empresas por setor;
* Comparar indicadores fundamentalistas;
* Identificar sinais de força, atenção ou risco;
* Apoiar uma análise inicial com base em dados.

---

## Funcionalidades do MVP

### 1. Base de empresas

Criar uma base com empresas brasileiras selecionadas e seus principais indicadores.

Indicadores considerados:

* P/L;
* ROE;
* ROIC;
* Margem líquida;
* Dividend Yield;
* Dívida líquida/EBITDA;
* Liquidez corrente.

---

### 2. Classificação das empresas

Criar uma classificação simples para apoiar a leitura dos dados:

* Forte;
* Atenção;
* Risco.

Essa classificação será baseada em critérios objetivos definidos a partir dos indicadores.

---

### 3. Consultas SQL

Criar consultas para responder perguntas como:

* Quais empresas têm maior ROE?
* Quais empresas têm maior Dividend Yield?
* Quantas empresas existem por setor?
* Quais empresas foram classificadas como Forte?
* Quais empresas apresentam sinais de risco?

---

### 4. Dashboard inicial

Criar uma visualização simples com:

* Cards de indicadores;
* Filtro por setor;
* Ranking de empresas;
* Gráficos comparativos;
* Tabela consolidada.

---

## Fora do escopo do MVP

Nesta primeira versão, o produto não terá:

* Atualização automática de dados;
* Login de usuário;
* Recomendação de compra ou venda;
* Integração com corretoras;
* Análise em tempo real;
* Inteligência artificial preditiva.

---

## Critérios de Sucesso

O MVP será considerado válido se permitir que o usuário:

* Compare empresas de forma simples;
* Entenda os principais indicadores;
* Identifique empresas com sinais positivos ou negativos;
* Reduza o esforço de análise;
* Explique por que uma empresa está classificada como Forte, Atenção ou Risco.

---

## Aprendizado Esperado

Com este MVP, o projeto busca validar se a organização visual dos dados fundamentalistas ajuda o usuário a tomar decisões mais conscientes e estruturadas.

O foco está em transformar dados técnicos em uma experiência mais clara, acessível e comparável.
