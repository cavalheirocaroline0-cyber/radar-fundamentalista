# Observações sobre a Base de Dados

A base `empresas_amostra.csv` foi criada com dados simulados para fins educacionais e de desenvolvimento do projeto.

O objetivo desta base é permitir a criação de consultas SQL, testes de classificação e construção inicial do dashboard.

Em uma versão futura, os dados poderão ser substituídos por informações reais extraídas de fontes públicas, como Fundamentus ou outras bases financeiras abertas.

Este projeto não representa recomendação de compra, venda ou manutenção de ativos.
